Design Notes & Style Guide (For Replit Implementation)

Overall Aesthetic:
Minimal & Modern: Clean, whitespace-heavy layout. Dark mode dominant (90% of site). Feels like Perplexity or Linear – professional, futuristic, no clutter.
Layout Structure:
Hero Section: Full-width dark BG with centered H1 ("Craft content that wins AI search"), subheadline, and gradient CTA button. Parallax subtle scroll effect.
Navigation: Fixed top bar (#0F172A BG), white text, indigo links. Logo left, menu items (Insights, Workflows, Pricing) right, "Book Demo" button prominent.
Sections: Card-based (white borders on dark BG), alternating left/right alignment. Testimonials as quote cards with green accents.
Footer: Dark BG, white links, social icons in indigo.


Typography:
Headings: Inter or SF Pro Bold (font-weight: 700), sizes 48px (H1), 32px (H2), line-height 1.2.
Body Text: Inter Regular (font-weight: 400), 16px, line-height 1.6. Use #F8FAFC on dark, #0F172A on light cards.
CSS Snippet for Replit:text@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;700&display=swap');
body { font-family: 'Inter', sans-serif; }
h1 { font-size: 48px; font-weight: 700; color: #F8FAFC; }
p { font-size: 16px; color: #94A3B8; }

Buttons & CTAs:
Primary CTA: Gradient indigo-green BG, white text, rounded (border-radius: 8px), hover: scale 1.05 + brighter gradient.
Secondary: Outlined (#6366F1 border), indigo text, subtle shadow.
CSS Example:text.cta-primary { background: linear-gradient(135deg, #6366F1, #10B981); color: white; padding: 16px 32px; border-radius: 8px; transition: transform 0.2s; }
.cta-primary:hover { transform: scale(1.05); }

Key Visual Elements:
Cards: Dark BG (#0F172A) with #1E293B borders, subtle box-shadow (0 4px 6px rgba(0,0,0,0.1)). Green accents for icons/checkmarks.
Icons/Graphics: Simple line icons in #6366F1 (e.g., from Heroicons). Subtle animations (fade-in on scroll).
Images: High-res screenshots of dashboards/workflows (dark-themed). No stock photos – use AI-generated or custom.
Responsiveness: Mobile-first: Hero stacks vertically, nav hamburger menu.

Replit Implementation Tips:
Start with: Next.js template (for SSR/SEO) or plain HTML/CSS/JS.
CSS Framework: Tailwind (e.g., bg-slate-900 text-white for dark BG).
Full Palette CSS:text:root {
  --primary: #6366F1;
  --secondary: #10B981;
  --bg-dark: #0F172A;

  --border: #1E293B;
}
